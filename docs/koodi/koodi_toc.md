# Koodihaasteet

Tällä sivulla on erilaisia kokeiluja siitä, miten Oulun yliopiston opetuskäytössä voitaisiin mahdollisimman helposti käyttää Advent of Code (tms) tyylisiä ohjelmointitehtäviä.
