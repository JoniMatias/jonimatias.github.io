# Glossaari


## Pakomerkki

Pakomerkki tarkoittaa tekstissä olevaa merkkiä, joka ohjeistaa sitä tulkitsevaa ohjelmaa lukemaan pakomerkin jälkeinen merkki tai merkkisarja eri merkityksessä kuin ruudulla näytettävänä kirjaimena. Yleensä tällaisia merkkejä käytetään, kun halutaan epätavallisia merkkejä tekstiin tai lisätä toimintalogiikkaa tavallisen tekstin sekaan.

Hyvä esimerkki pakomerkistä on mm. C-ohjelmointikielen printf-funktion kenoviiva ```\```, joka muuttaa mm. merkkisarjat ```\n``` rivin vaihdoksi, ```\t``` tabiksi ja ```\"``` lainausmerkiksi.

Bash-komentotulkki käyttää myös kenoviivaa pakomerkkinä. Yleisin käyttötapaus bashissa on muuttaa argumentit erottava välilyönti argumentin sisäiseksi välilyönniksi pakenemalla välilyönnin merkkisarjalla ```\ ``` (kenoviiva ja välilyönti).
