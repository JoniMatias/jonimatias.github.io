# Glossaari


## Pakomerkki

Pakomerkki tarkoittaa tekstissä olevaa merkkiä, joka ohjeistaa tekstiä tulkitsevaa ohjelmaa lukemaan pakomerkin jälkeinen merkki tai merkkisarja eri merkityksessä kuin se muuten olisi. Yleensä tällaisia merkkejä käytetään, kun halutaan epätavallisia merkkejä tekstiin. Toinen yleinen käyttötapaus pakomerkeille on lisätä toimintalogiikkaa tavallisen tekstin sekaan.

Yleisiä esimerkkejä pakomerkistä on mm. C-ohjelmointikielen printf-funktion kenoviiva ```\```, joka muuttaa mm. merkkisarjat ```\n``` rivin vaihdoksi, ```\t``` tabiksi ja ```\"``` lainausmerkiksi.

Bash-komentotulkki käyttää myös kenoviivaa pakomerkkinä. Yleisin käyttötapaus bashissa on muuttaa argumentit erottava välilyönti argumentin sisäiseksi välilyönniksi pakenemalla välilyönnin ```\ ``` (kenoviiva ja välilyönti).
