# Ohjausrakenteet

Komentorivin ohjausrakenteet toimivat käytännössä samalla tavalla kuin ohjelmointikielten ohjausrakenteet. Jos ohjelmointikielten ohjausrakenteet eivät ole tuttuja, niin ne kyllä tulevat tutuksi Ohjelmointi 1 -kurssilla. 

Ohjausrakenteiden tarkoitus on muuttaa ohjelman kulkua aina, kun jokin ehto täyttyy. Ohjelman kulku voi käytännössä muuttua kahdella tavalla: haaroilla tai silmukoilla. Haarat joko suorittavat tai jättävät suorittamatta joitain komentoja annetun ehdon mukaisesti. Silmukat taas toistavat komentojaan niin kauan kun annettu ehto on totta.

Kuten tavallista, Microsoft on kehitellyt Windowsille täysin omanlaisen tyylin ohjausrakenteille. Tämän vuoksi alla on linkit <nowrap>:os-win: .bat</nowrap>-tiedostojen ohjausrakenteisiin ja <nowrap>:os-mac::os-linux: bashin</nowrap> Unix-standardeja noudattaviin ohjausrakenteisiin.
