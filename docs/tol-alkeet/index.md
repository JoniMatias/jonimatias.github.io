# TOL-alkeet

TOL-alkeet osiosta löytyy paljon hyvää ja hyödyllistä Oulun yliopiston tietojenkäsittelytieteen opiskelijalle. Sivusto alkaa hyvin yksinkertaisista tietokoneeseen liittyvistä asioista, kuten sähköpostista ja tiedostoista. Nämä ovat yksinkertaisia asioita, mutta jokaisen opiskelijan on hyvä ne tietää.

Alkeiden jälkeen päästään käsittelemään komentorivin käyttöä. Tietojenkäsittelyn opintojen aikana komentorivin käyttö tulee jossain vaiheessa vastaan, joten sen käyttö kannattaa opetella mahdollisimman varhain. Kyllä opinnoista selviää ilman komentoriviäkin, mutta komentorivi on sen verran käytännöllinen työkalu, ettei sitä kannata jättää käyttämättä.

Vasemmasta reunasta voit navigoida haluamaasi osioon helposti.


Tämän sivun sisältö on pitkälti [MIT:n missing semesterin](https://missing.csail.mit.edu/) innoittama. Jos haluat perehtyä asiaan enemmän, suosittelen käymään kyseisen sivuston läpi.
